# 420 Worlds - Prototype

**Live Demo:** https://laibyrinth.com/420/

## What is it?
A Roblox-inspired social app built around the 4:20 timezone experience. Users create customizable avatars, join timezone-based worlds, check in at 4:20, earn weed points, and unlock cosmetics.

## Features (Current Build)
- Age gate (21+ verification)
- Avatar creator (skin tones, hair, outfit colors)
- Main hub with countdown to next 4:20
- 12 timezone worlds (NYC, LA, Amsterdam, Tokyo, etc.)
- Real-time world status (shows which are live at 4:20)
- World rooms with avatars + chat
- Check-in system (earn 420 weed points per check-in)
- Daily streak tracking
- Drip store (20+ unlockable items)
- Profile with stats and badges

## Tech Stack
- **Frontend:** Single HTML file with inline CSS/JS
- **State:** localStorage (no backend yet)
- **Fonts:** Google Fonts (Poppins, Rajdhani)
- **No dependencies** - runs entirely client-side

## Run Locally
1. Clone this repo
2. Open `index.html` in a browser
3. That's it - no build process needed
